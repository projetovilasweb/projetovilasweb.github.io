Tutorial

Adicionar termos ou expressões na string

Para adicionar o termo ou expressão de sua escolha, clique na caixa com os dizeres "Insira uma expressão ou um termo aqui", no alto da tela e pressione o botão inserir, ou a tecla enter

Adicionar operadores

A adição de um operador se dá clicando no botão correspondente a ele, logo acima do campo de visualização de strings

Excluir elementos da string

Para remover qualquer elemento de sua string. basta clicar duas vezes sobre ele

Edição da string

Qualquer termo ou elemento da string pode ter sua ordem alterada, clicando uma vez com o mouse sobre ele, arrastando até a posição desejada e soltando o botão do mouse

Validação e visualização

Após a geração da string, é necessário fazer sua validação, para garantir que ela possui sentido. Então, deve-se clicar no botão de validação de string. O campo logo abaixo desse botão, dada a string válida, é responsável por mostrar a string atual gerada, que será convertida para os mecanismos de busca requeridos.

Buscas

Para buscar em qualquer base de dados, deve-se clicar nos botões correspondentes, abaixo da string validada
